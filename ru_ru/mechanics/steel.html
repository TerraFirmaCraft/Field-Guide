Сталь

Сталь — это продвинутый материал, который можно использовать для создания инструментов и брони. Существует несколько ее разновидностей: сталь, чёрная сталь, красная сталь и синяя сталь.

Чтобы создать сталь, сначала нужно создать передельный чугун в доменной печи и отлить его в слитки.

Уровень 0

Слиток передельного чугуна может быть обработан на наковальне для создания слитка высокоуглеродистой стали, который, в свою очередь, может быть снова обработан для создания слитка стали.

Чёрная сталь

Чёрная сталь является улучшенной формой стали, получаемой из сплава стали и других металлов. Для её создания вам потребуется создать сплав слабой стали в тигле.

Слабая сталь


Требования:

  • 50–70 %: Сталь
  • 15–25 %: Никель
  • 15–25 %: Чёрная бронза


Расплавленную слабую сталь можно отлить в слитки.

Рецепт: tfc:welding/metal/ingot/high_carbon_black_steel

Слитки слабой стали затем можно сварить с передельным чугуном, чтобы создать слитки высокоуглеродистой чёрной стали. Наконец, их можно обработать на наковальне, чтобы получить слитки чёрной стали.

Чёрная сталь может использоваться для создания инструментов и брони, а также является ключевым ингредиентом в создании синей и красной сталей.

Синяя сталь

Синяя сталь является одним из двух металлов высшего уровня вместе с красной сталью. Подобно чёрной стали, первым шагом является создание сплава слабой синей стали в тигле.

Слабая синяя сталь


Требования:

  • 50–55 %: Чёрная сталь
  • 20–25 %: Сталь
  • 10–15 %: Висмутовая бронза
  • 10–15 %: Стерлинговое серебро


Рецепт: tfc:welding/metal/ingot/high_carbon_blue_steel

Слитки слабой синей стали можно сварить сс слитками черной стали, чтобы создать слитки высокоуглеродистой синей стали. Наконец, их можно обработать на наковальне, чтобы создать слитки синей стали.

Синяя сталь может использоваться для создания инструментов и брони, а также ведра из синей стали, которое может использоваться в том числе для транспортировки лавы.

Красная сталь

Красная сталь является одним из двух металлов высшего уровня вместе с синей сталью. Подобно чёрной стали, первым шагом является создание сплава слабой красной стали в тигле.

Слабая красная сталь


Требования:

  • 50–55 %: Чёрная сталь
  • 20–25 %: Сталь
  • 10–15 %: Латунь
  • 10–15 %: Розовое золото


Рецепт: tfc:welding/metal/ingot/high_carbon_red_steel

Слитки слабой красной стали можно сварить со слитками черной стали, чтобы создать слитки высокоуглеродистой красной стали. Наконец, их можно обработать на наковальне, чтобы создать слитки красной стали.

Красная сталь может использоваться для создания инструментов и брони, а также ведра из красной стали, которое может использоваться в том числе для транспортировки лавы.
