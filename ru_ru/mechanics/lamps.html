Лампы

Лампы — это долгоиграющий источник света. Они используют жидкое топливо. Лампы сохраняют своё содержимое при разборке. Используя ведро с топливом, нажмите ПКМ на лампе, чтобы добавить в неё. Затем лампу можно зажечь с помощью огнива или чего-либо, способного поджигать. Оливковое масло и жир — стандартное топливом для ламп. Выстрел из лука по зажжённой лампе может вызвать пожар.

Лампы

Структура

Зажжённая лампа, висящая на цепи.

Лампы куются на наковальне и объединяются со стеклом для лампы.

2

Одним из видов топлива для ламп является оливковое масло. Первым шагом в его производстве является изготовление оливковой пасты.

Вам также понадобится сеть из джута.

Вскипятите оливковую пасту с водой в горшке, чтобы сделать воду с оливковым маслом. Запечатайте это в бочке с сетью из джута, чтобы получить оливковое масло. Оливковое масло горит 8 игровых часов на каждую единицу жидкости.

Другим видом топлива для ламп является жир. Чтобы сделать его, растопите 5 жира в горшке с водой. Жир горит меньше 2 игровых часов на единицу. Его можно использовать для приготовления свечей
