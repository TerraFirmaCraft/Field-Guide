Устройства для печей

Печи имеют ряд устройств, которые взаимодействуют с ними и расширяют их функциональность. Это связано с тем, что печи по своей природе модульны.

Бункер для печи будет загружать бревна в любую нижнюю печь, на которую он направлен. Он вмещает 16 бревен (4 стека по 4, как штабель бревен), и его инвентарь наполняется путем бросания предметов сверху. Он также может получать питание с помощью автоматизации из других модов.

Пепельница собирает древесный пепел, когда размещается под нижним блоком печи. Есть 0,5 вероятности, что она получит пепел при сгорании топлива. Пепел извлекается с помощью ПКМ и вставляется с помощью атаки.

Рецепт: firmalife:crafting/vat

Чан производит некоторые избранные рецепты кипячения оптом. Он имеет одно место для предметов и 10 000 мБ пространства для жидкости, аналогично бочке.

Например, в чане можно приготовить оливково-масляную воду, используя соотношение 1 оливковая паста к 200 мБ воды. Чтобы использовать чан, ПКМ его с жидкостями и предметами, чтобы добавить их в инвентарь. С пустой рукой и зажатой клавишей Shift, щелкните, чтобы закрыть и открыть чан. Чан не будет кипятить, пока он не будет закрыт.

Vats should be placed on the block above a Bottom Oven. If the vat would overflow on completion of the recipe, it will not boil, so be sure not to overfill it -- especially with recipes that produce more fluid than they consume! Vats can be opened and closed with Redstone.

Структура

Рецепт: firmalife:crafting/jarring_station

Станция для закатывания используется для заполнения банок из чана. Станция для закатывания имеет носик с одной стороны, который должен быть направлен на станцию.

С помощью чана можно приготовить сахаристую воду, добавив подсластитель в воду. Затем, добавление фруктов приводит к приготовлению фруктовой жидкости. Щелчок по ней с помощью пустой банки с крышкой заполнит её. Будьте осторожны: попытка перекачать или транспортировать фруктовую жидкость через слишком много перекачек может испортить её!

С помощью ПКМ добавьте пустые банки с крышками на станцию для закатывания. Когда она обнаружит фруктовую жидкость, она автоматически заполнит банки вареньем и закроет их, требуя 500 мБ фруктовой жидкости на банку.

Горшки и грили из TFC можно размещать сверху нижней печи. Эти устройства будут автоматически получать тепло от нижней печи. Горшок можно использовать только для приготовления супа. Он не может выполнять обычные рецепты для горшка.
