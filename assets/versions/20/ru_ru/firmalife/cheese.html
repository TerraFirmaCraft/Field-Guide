Сыр

Изготовление сыра в Firmalife немного сложнее, чем в ванильном TFC. Есть два новых вида молока: молоко яка и козье молоко. Их получают, доя яка и козу соответственно. Дойка коровы по-прежнему дает старый вид молока.

Как обычно, молоко сначала должно быть свернуто. Для свертывания молока вам нужен сычуг. Сычуг получают из желудка жвачных животных. К ним относятся яки, коровы, овцы, козы и мускусные быки. Для свертывания молока запечатайте его в бочке с сычугом на 4 часа.

8

Свернувшееся молоко (простокваша) должна быть преобразована в творог, запечатав его в бочке с марлей. Марля не многоразовая.

Вы готовы сделать твердый сыр, если хотите. Вы можете сделать Райя-Меток из творога яка, Шевр из козьего творога и Чеддер из коровьего творога.

Другой вариант — сделать мягкие сыры. Они изготавливаются путем запечатывания творога в бочке с соленой водой. Вы можете сделать Шоша из творога яка, Фета из козьего творога и Гауда из коровьего творога.

Головки сыра — это блоки, которые следует размещать для увеличения срока их хранения. Для улучшения качества и срока годности головки сыра следует выдерживать в погребе. Чтобы получить съедобный сыр из головки сыра, его следует срезать с головки, щелкнув ПКМ с помощью ножа. Если блок просто сломать, выдержка теряется!
