Лотки для рассады

Лотки для рассады используются для выращивания культур внутри теплицы. Чтобы увидеть состояние лотка, вы можете посмотреть на него, держа в руках мотыгу. Культуры в лотках потребляют питательные вещества аналогично культурам. Лотки должны быть размещены внутри действующей теплицы и активированы с помощью климатической станции. Лоткам необходимо хотя бы некоторое количество естественного солнечного света для работы.

Лотки для рассады должны быть увлажнены для роста. Это делается с помощью лейки, изготовленной из деревянного ведра, емкости с водой и древесины. Нажмите ПКМ с ней, чтобы полить близлежащие лотки. Заполните ее, нажав ПКМ на источнике воды.

Большие лотки — самый простой тип лотков. Они выращивают один урожай из семян и собираются с помощью ПКМ после созревания.

В больших лотках можно выращивать зеленую фасоль, томаты, сахарный тростник, джут и зерна. Однако для выращивания зерна вам нужна теплица из меди или лучше.

Лоток для небольших растений выращивает четыре отдельных культуры одновременно. Все эти культуры используют один и тот же резервуар питательных веществ и могут собираться по отдельности с помощью ПКМ после созревания.

В лотках для небольших растений можно выращивать свеклу, капусту, морковь, чеснок, лук, картофель и сою. Эти культуры можно выращивать в любом типе теплицы.

Лоток для бонсая выращивает небольшие плодовые деревья из их саженцев. Плоды можно собирать с помощью ПКМ.

В лотках для бонсая можно выращивать любые виды плодовых деревьев, кроме бананов, которым нужен подвесной лоток. Все они потребляют азот в качестве основного питательного вещества. Для их выращивания необходима теплица из железа или лучше.

Подвесные лотки выращивают культуры вверх ногами. После созревания их можно собирать с помощью ПКМ.

В подвесных лотках выращивают тыкву из семян и бананы из саженцев. Тыкву можно выращивать в любой теплице, но для бананов требуется теплица из железа или лучше. Подвесные лотки должны крепиться к твердому блоку над ними.

Подпорки для кустов выращивают ягодные кусты. Ягоды можно собирать с помощью ПКМ.

Подпорки для кустов обладают уникальным свойством размножения кустов ягод. Если в подпорки для кустов поместить на другой, и нижний имеет зрелый куст ягод, у него есть шанс вырасти вверх в следующий. Подпорки для кустов могут выращивать любые кусты ягод, кроме клюквы, но для работы им требуется теплица из железа или лучше. Кусты предпочитают азот.

Hydroponic Planters grow rice and cranberry bushes. They work the same as a quad planter, except that they do not need to be watered.

Hydroponic Planters must instead, have a

  • Sprinkler Pipe below them that is supplying water. Without the pipe they will not grow. There is no hard limit to how many hydroponic planters may be fed by one station.

Рецепт: firmalife:crafting/sweeper

Sweepers are used to harvest planters other than Quad and Hydroponic planters. They must be connected to a rotating Axle on their bottom face.

Рецепт: firmalife:crafting/picker

Pickers can be powered via redstone signal from above. They extend down to pick the crops from Quad and Hydroponic planters.
