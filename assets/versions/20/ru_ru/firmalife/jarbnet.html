Стеллажи для банок

Стеллаж для банок — это декоративный блок для хранения банок, свечей и кувшинов. Его можно открывать и закрывать, щелкнув пустой рукой и нажав Shift. Если внутри находятся свечи, его можно зажечь, чтобы получить небольшое количество света.
